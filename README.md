# chatbot_with_memory
Desenvolvi um chatbot com IA utilizando o modelo da DeepSeek integrado a uma interface web construída com Streamlit.  O projeto foi criado com foco em interação conversacional simples, responsiva e funcional, permitindo que usuários conversem com a IA em tempo real através de uma interface intuitiva.  Além das funcionalidades básicas de chat, implementei um sistema de memória contextual, permitindo que o chatbot mantenha informações da conversa ao longo da interação, tornando as respostas mais naturais e coerentes.  O projeto também envolveu integração com APIs, manipulação de contexto de mensagens, estruturação de prompts e organização de aplicação full Python para desenvolvimento de soluções com IA.
